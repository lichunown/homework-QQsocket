
1、安装gtk教程：
  http://blog.csdn.net/rekols/article/details/43864549
2、编译含有gtk文件的命令
    例： gcc -o main main.c `pkg-config --cflags --libs gtk+-2.0`
3、界面功能说明：
  （1）注册
       需要id、密码、确认密码
  （2）登录
      需要id、密码
  （3）添加好友
  （4）双击好友名字打开对话框
  （5）删除好友，功能按钮出现在好友对话框中
4、未完成的功能
   （1）注册功能和服务器的交互
    （2）登录功能和服务器的交互
    （3）添加好友和服务器的交互
    （4）删除好友和服务器的交互
    （5）聊天信息和服务器的交互
5、目前可以运行的测试功能
   （1）程序运行后进入登录界面，可以直接点击登录进入主界面
   （2） 进入主界面可以下拉好友列表，双击好友名字进入对话框（好友是在程序中提前加进去做测试用，未和服务器交互）
   （3） 进入聊天框可发送消息到聊天框;
6、存在的问题
   没有做多线程，打开两个对话框程序就由可能崩溃
7、运行方法：
   gcc -o main main.c `pkg-config --cflags --libs gtk+-2.0`
   ./main
